# Projeto Final da Unidade Curricular de Implantação de Sistemas

## Portal de Solicitação de Peças Internas 
Colaboradores das fábricas solicitam peças por e-mail, gerando atrasos e falta de rastreabilidade.

## Desafio
criar um portal web para solicitação, aprovação e acompanhamento de pedidos internos, com controle de estoque integrado.
